🎮 Twitch OAuth Generator

Generate OAuth tokens for Twitch Kodi Addon

Before you start:
You need a Twitch Application. Create one at dev.twitch.tv/console/apps
1
Go to Twitch Developer Console and create a new application (or use an existing one)
2
Set the OAuth Redirect URL to:
https://YOUR_GITHUB_USERNAME.github.io/plugin.video.twitch/docs/oauth/callback.html
3
Copy your Client ID from the application settings and paste it below
⚠️ Security Note: Your Client ID is only stored locally in your browser and is never sent to any server except Twitch during authentication.

✅ OAuth Token Generated!
